# CIC Project Lab v5 — Full Lifecycle

This is a full static browser-based MVP for a Clinical Implementation Consultant onboarding simulator.

## How to run

1. Unzip this folder.
2. Open `index.html` in a browser.
3. Work through all 14 project phases.
4. Ask stakeholder interview questions.
5. Build deliverables.
6. Complete configuration exercises.
7. Work through go-live war room issues.
8. Complete the patient readiness certification board.

No server required.

## Included

- Full 14-phase lifecycle
- Project health dashboard
- Living risk register
- Stakeholder interviews and hidden findings
- Consultant decision engine
- Senior CIC mentor feedback
- Deliverable builders
- Configuration exercise
- Interface testing scenario
- EMR go-live war room
- Patient readiness board
- Treatment go-live certification scoring
- JSON-driven scenario data in `data/scenario.json`

This version removes the specific cancer center name and uses **CIC Cancer Center** as the fictional training customer.

## Scenario template layer

The scenario library now includes four reusable onboarding levels:

1. CIC Cancer Center Community — foundational
2. CIC Cancer Center Regional — intermediate
3. CIC Cancer Center University — advanced
4. CIC Cancer Center Network — expert / consolidation

The current full lifecycle simulation uses the Regional-style hospital-owned scenario with Epic, ARIA, Eclipse, interoperability, EMR training, EMR go-live, and treatment go-live.
